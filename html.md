# HTML basics

**Wireframe**

start off with a wireframe design, draw what you want your website to look like with MSpaint or preferably a more proffessional app.

**Basic code**
```
<!DOCTYPE html>
<html>
<h1>this is a header</h1>
<body>
<p>this is a sentence</p>
</body>
</html>
```
